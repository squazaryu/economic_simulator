# Экономический симулятор сценариев для рынка РФ

Интерактивный Streamlit-проект для моделирования влияния макрофакторов на:
- индекс **IMOEX**,
- выбранную **акцию MOEX** (тикер вводится пользователем).
- **портфель MOEX** (2-5 акций с весами).

## Ключевые возможности

- Загрузка реальных данных за последние 5 лет (месячная частота).
- Прогноз по сценарию для IMOEX и отдельной акции.
- Прогноз по сценарию для портфеля акций.
- Расширенные сценарные параметры:
  - Brent,
  - ключевая ставка,
  - USD/RUB,
  - инфляция,
  - рыночный сентимент,
  - ликвидность,
  - геополитический фактор,
  - регуляторный фактор,
  - масштаб волатильности для Monte Carlo.
- Monte Carlo (10 000 сценариев) с P5/P50/P95 и вероятностью падения >20%.
- Sobol-анализ чувствительности (S1) и Tornado Chart.
- Подробные пояснения на каждой вкладке интерфейса.
- Режимы модели: весь период, до февраля 2022, с февраля 2022.
- Динамический веерный прогноз на 6-12 месяцев (траектория цены/индекса, P10-P90).
- Side-by-side сравнение сценариев: базовый / оптимистичный / пессимистичный.
- Авто-комментарий по драйверам прогноза.

## Источники данных

1. **ЦБ РФ**
- `cbrapi`: ключевая ставка, USD/RUB.
- Официальная таблица ЦБ: инфляция `https://www.cbr.ru/hd_base/infl/`.

2. **MOEX ISS API**
- История IMOEX.
- История выбранной акции MOEX.

3. **Yahoo Finance (`yfinance`)**
- Brent (`BZ=F`).

## Установка и запуск

Требуется Python 3.11+

```bash
cd economic_simulator
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## Логика модулей

1. `src/data_loader.py`
- Загрузка CBR/MOEX/Brent.
- Сохранение в `data/raw/`.
- Fallback на локальные CSV при недоступности API.
- Поддержка загрузки произвольного тикера MOEX.

2. `src/preprocessing.py`
- Очистка и нормализация дат.
- Интерполяция и заполнение пропусков.
- Сохранение итогового датасета в `data/processed/`.

3. `src/model.py`
- Отдельная модель для IMOEX.
- Отдельная модель для выбранной акции.
- Режимные версии моделей (all/pre_2022/post_2022).
- Метрики `R²`, `MAE`, `RMSE`.
- Сценарные поправки (сентимент/ликвидность/геополитика/регуляторика).
- Декомпозиция драйверов прогноза.

4. `src/monte_carlo.py`
- Симуляция неопределенности по макрофакторам.
- Поддержка IMOEX, выбранной акции и портфеля.
- Учет режимов модели и дополнительных поправок.

5. `src/sensitivity.py`
- Sobol-анализ с Saltelli sampling.
- Tornado Chart с сортировкой факторов по S1.
- Поддержка IMOEX/акции/портфеля и режимов модели.

6. `app.py`
- 4 вкладки: история, сценарий, Монте-Карло, чувствительность.
- Подробный текст-описание на каждой вкладке.
- Полная русификация пользовательского интерфейса.
- Динамический график будущей траектории и веер неопределенности.
- Сравнение сценариев и портфельный режим.

## Ограничения

- Линейная регрессия отражает только линейные зависимости.
- Дополнительные сценарные поправки применяются как пост-коррекция прогноза.
- При сбоях API используются последние локально сохраненные срезы.
